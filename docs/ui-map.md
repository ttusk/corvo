# Corvo UI Map

## Main View

Corvo uses a single Obsidian `ItemView` with internal tabs.

The view is opened by:

- ribbon icon
- `Corvo: Open Corvo`

## Screens

### Dashboard

- active contest
- current subject
- next subject
- quick actions for cycle advance and quick study sessions
- summary by subject
- progress overview

### Concursos

- create contest
- list contests
- switch active contest
- active contest notes preview

### Ciclo e Materias

- create subject
- ordered subject list
- move up/down
- activate/deactivate
- update planned minutes
- update current stage

### Itens e PDFs

- subject picker
- create study item
- show PDF progress by item
- attach item-level resource references

### Assuntos e Questoes

- subject picker
- create topic
- attach topic-level resource references
- link Tec Concursos notebook
- show notebook counters

### Sessoes

- register manual sessions for `pdf`, `video`, and `questions`
- select subject, item, and topic
- view recent session history

### Mural

- edit notice link
- edit exam link
- edit wall notes
- show wall summary
